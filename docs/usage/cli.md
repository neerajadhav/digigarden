# Usage


The general command line syntax for Zensical is:

```sh
zensical COMMAND [OPTIONS] [ARGS]...
```

## Commands

- [`new`](new.md)
- [`build`](build.md)
- [`serve`](preview.md)

## Help

- General help: `zensical --help`
- Command-specific help: `zensical <command> --help`
