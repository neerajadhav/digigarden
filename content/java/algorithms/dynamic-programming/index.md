# Dynamic Programming

Dynamic Programming (DP) is an algorithmic technique for solving complex problems by breaking them down into simpler subproblems. It is used when subproblems overlap and have optimal substructure.

## Topics
- **[Fibonacci Sequence](01-fibonacci.md)** - Re-solving Fibonacci using Memoization, Tabulation, and Matrix Exponentiation.
- **[Longest Common Subsequence](02-longest-common-subsequence.md)** - Finding the longest shared subsequence between two strings.
- **[Longest Increasing Subsequence](03-longest-increasing-subsequence.md)** - Finding the length of the longest sorted subsequence.
- **[0/1 Knapsack](04-knapsack.md)** - Solving the classic optimization problem using DP.
- **[Coin Change](05-coin-change.md)** - Finding the minimum coins to reach a target sum.
- **[Matrix Chain Multiplication](06-matrix-chain-multiplication.md)** - Optimizing matrix multiplication order using the partitioning pattern.
