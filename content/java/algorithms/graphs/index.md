# Graph Algorithms

Graph theory is the study of graphs, which are mathematical structures used to model pairwise relations between objects. Graph algorithms are used in many applications, from social networks to GPS navigation.

## Topics
- **[Breadth-First Search (BFS)](01-bfs.md)** - Exploring a graph level by level.
- **[Depth-First Search (DFS)](02-dfs.md)** - Exploring a graph branch by branch.
- **[Dijkstra's Algorithm](03-dijkstra.md)** - Finding the shortest path in weighted graphs.
- **[Floyd-Warshall Algorithm](04-floyd-warshall.md)** - Finding all-pairs shortest paths in weighted graphs.
- **[Topological Sort](05-topological-sort.md)** - Ordering vertices in a DAG based on dependencies.
- **[Kruskal's Algorithm](06-kruskals.md)** - Finding the Minimum Spanning Tree (MST) of a graph.
- **[Prim's Algorithm](07-prims.md)** - Finding the Minimum Spanning Tree (MST) using a node-based approach.
